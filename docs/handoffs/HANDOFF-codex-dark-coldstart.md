# Handoff · Codex:冷启动屏深色 v2(逐屏铺开 · 第2屏)

> **背景**:轨迹页深色 v2 已验证定稿,tokens 确立。现逐屏铺开,本次做**冷启动屏**(第一眼 + 表单重灾区)。复用 `FRONTEND-tokens-v2-dark.md`(已定稿色板)。
> **⚠️ 前置**:等轨迹页微调完成后再做本屏(保持小步)。
> **红线**:纯前端视觉。不动后端/eval/表单逻辑(必填校验、字段、E7c 未知补位都保留,只改视觉)。分支 `frontend-replit-v1`。
> **目标**:解决"表单无趣"——用深色 + 冷蓝交互 + 精致选中态,把一堆灰框变成有科技仪器感的"校准录入"。

---

## 1. 应用深色 tokens(同轨迹页)

- 底 `--void`、卡 `--surface`、线 `--line`、文本 `--text`/`--text-dim`
- **全站冷光晕**:同轨迹页那套(统一位置逻辑,视觉锤一致性),氛围底层,表单永远清晰在其上

## 2. 表单选中态(核心,解决"无趣"+ 之前的选中无反馈)

用交互蓝 `--interact` `#3D7BD9`:
- **选中项**:`--interact` 冷蓝描边(1.5-2px)+ 极淡冷蓝内发光 + radio 圆点填充冷蓝
- 选中背景:极淡冷蓝调(`--interact-deep` `#0A4CA2` 的极低透明度)或 `--surface-raised`
- **未选**:`--surface` 底 + `--line` 细边,中性
- 参照 Chillin 给的 ChatGPT app 深色图:选项卡的青绿描边发光感(但用我们的冷蓝)
- **⚠️ 不用 `--converge` 绿**:绿是收敛专用,表单交互用蓝

## 3. 表单层次(解决"一堆同款框")

- **字段标签**(烘焙度/粉量/磨豆机/研磨):等宽或半等宽,`--text-dim`,仪器标签感
- **分组留白节奏**:字段之间留白有呼吸,不是挤成一列同款框
- **选项卡片**:深色 + hairline 边 + 微妙质感(极淡边缘发光,克制)
- 保留 E7c 未知补位("不知道"/"不确定")作为合法选项,视觉上和普通选项一致(别弱化,它是合法选择)

## 4. 提交按钮 + 必填提示(深色版)

- **提交按钮**:深色下的主按钮。可用 `--interact` 冷蓝(可交互强调)或中性亮。⚠️ 不用收敛绿(那是收敛时刻,提交不是收敛)
- **必填未选提示**(之前 P2):克制标记,深色下用 `--interact` 冷蓝描边提示漏选项(不用红色报错),点灰按钮滚动/高亮到漏选项

## 5. 标题区(呼应"仪器校准"叙事)

- "冷启动录入" + "告诉仪器你的初始状态,以建立校准基线"——文案已是仪器叙事,视觉深色科技感呼应它
- 标题区可略加质感(等宽副标题、微妙分隔),强化"这是个精密仪器"的第一印象

---

## 6. 分层密度(触点屏=疏)

冷启动是**触点屏**(初学者第一次接触),按 tokens 规格走**疏、留白**,别做太密。让初学者放松,不是仪表盘那种密。深色 + 疏 + 精致选中,气质是"高级但不吓人"。
