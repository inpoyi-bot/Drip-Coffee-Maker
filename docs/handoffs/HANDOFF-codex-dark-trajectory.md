## 1. 应用深色 tokens(全局基础)

按 `FRONTEND-tokens-v2-dark.md`:
- 页面底 `--void` `#0E1013`(深炭微冷,非纯黑)
- 卡片 `--surface` `#16181D`;抬升/选中 `--surface-raised` `#1E2128`
- 主文本 `--text` `#E4E8ED`(近白微冷);次级 `--text-dim` `#7A828E`
- 分隔线 `--line` `#2A2E37`(hairline)

## 2. 全站冷光晕(视觉锤)

- 背景加 `--glow-cool`:用收敛绿/交互蓝**同色相的极暗弥散径向渐变**
- **红线**:极暗、极弥散(背景呼吸感,不是显眼色块);数据/文字永远清晰在其上;统一位置逻辑(建议中心偏上或某一致方位弥散)
- 参照 Chillin 给的深色参考图那种"几乎看不清的冷光笼罩"

## 3. 轨迹折线图(本屏核心,对标参考图3)

这是护城河高潮屏,重点做:

**折线本身**:
- 爬升段:中性色线(`--text-dim` 类),不抢戏
- **收敛点(最后一杯)**:`--converge` `#2ECC8F`(提亮版)实心点 + **发光光晕(glow)**,全页唯一彩色焦点。参照图3 高亮点的发光感。
- 线到收敛点可以从中性渐变到收敛绿(接近收敛处泛绿)

**区域填充(新增,解决"太空")**:
- 折线下方加**区域填充**:用 `--converge-deep` `#1B6F4C`(Chillin 原墨绿,暗版)做渐变/点阵填充,上浓下透明淡出
- 这就是"亮线 + 同色相暗填充"——`#2ECC8F` 亮点 + `#1B6F4C` 填充,一个色相两明度出深度
- 填充让图不再是"一条孤线 + 大片空白"

**坐标系(仪器感)**:
- 淡横向网格线(`--line` 极淡),给数据坐标感
- 横轴从第 1 杯开始(修过的,保持),等宽字体标刻度(仪器感)
- 纵轴若有刻度也等宽

**深色卡**:
- 整个图表放在 `--surface` 深卡里,可加极微妙边缘发光(克制)
- 数据线在深底上发光更突出

## 4. 其他元素深色适配(轨迹页内)

- 顶部"当前状态"摘要块:深卡 + 等宽数据
- 卡片列表(正序 + 当前摘要,已做的结构保留):深色化,状态标签用深色 chip
- 收敛那张卡(当前杯/已收敛):可用 `--converge` 点缀强调(它是收敛态),但克制——teal 仍是稀缺焦点
- 副标题"这些杯子来自你保存的记录,不是当前对话上下文":深色下可略提存在感(护城河陈述)

## 5. 收敛色纪律(v1 保留,深色强化)

- `--converge` `#2ECC8F` **仅**用于收敛相关(收敛点、已收敛态)。轨迹页里非收敛的杯、普通元素**不许**用它。
- 深底下 teal 更亮更诱人,越要守唯一性——满屏发光绿就廉价了。

## ⚠️ 给 Chillin 的验色说明
出样张后在真实屏幕看:
- **收敛点 `#2ECC8F` 不够亮/陷进背景** → 再提亮(如 `#34E0A0`)

- 光晕太强抢数据 → 压弱;太弱没氛围 → 提一点
